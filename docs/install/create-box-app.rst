Create Box App
==============

.. toctree::
   :maxdepth: 2
   :caption: Contents:
   
   
* Log into Box Developer Console
* Create New App
	* Select Custom App.
	* Select Server Authentication (Client Credentials Grant).
	* Enter the App Name in input box provided.
	* Click Create App Button
	* The application will open to the Configuration Window
		* Document the Client ID
		* Document the Client Secret
	* Open the General Settings
		* Document the Enterprise ID
	* Select App Access Only under App Access Level
* Authorize Application
	* Under the Authorization Section, click Review and Submit
* Log into Box Admin Console
	* Select the Apps Section
	* Select Custom Apps Manager
	* Select the Pending Authorization
	* Select Authorize App
